# 🤖 AI Chatbot (Rule-Based)

## 📌 Project Title
AI Chatbot - Rule-Based System (DecodeLabs Project 1)

---

## 📖 Description
This is a simple rule-based AI chatbot built using Python.  
It responds to predefined user inputs such as greetings, questions about AI, time, weather, and more.

The chatbot uses basic `if-elif-else` logic to match user intents and return appropriate responses.  
It also includes a fallback response for unknown inputs.

---

## ⚙️ Features
- Greeting detection (hi, hello, hey)
- Basic Q&A (name, AI, weather, time)
- Help menu
- Exit commands (exit, quit, bye)
- Fallback response for unknown inputs
